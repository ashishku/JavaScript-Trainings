• How a rendering engine works ....
  • User enters a new url in address bar.
  • Browser fetches the url.
  • Browser will cache the response.
  • Browser parses the response.
  • If browser sees a resource (i.e image, css or JavaScript) it will stop the parsing and will fetch the resource *>.
  • Once all resources are fetched and parse browser will create a document model tree.
  • Once tree is ready browser does the flow. In flow process it will calculate the positions and dimensions of individual elements of tree.
  • Browser will genrate a list of element wchich it needs to be displayed.
  • Browser will start painting the screen.
  • Finally indivual pixels will be genrated.
