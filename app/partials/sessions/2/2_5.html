Conditional Statements

Conditional statements execute or skip other statements depending on the value of a specified expression. These statements are the decision points of your code, and they are also sometimes known as “branches.”
  • if 
      if (expression)
    statement

  if (username == null)         // If username is null or undefined,
    username = "Ashish";        // define it

  // If username is null, undefined, false, 0, "", or NaN, give it a new value
  if (!username) username = "Ashish";
  • if else 
      if (expression)
    statement1
  else
    statement2
  
  if (n == 1)
    console.log("You have 1 new message.");
  else
    console.log("You have " + n + " new messages.");
  • else if
    There is no else if statement in JavaScript. It is rather a clever combination of if statement in place of a else statement.
