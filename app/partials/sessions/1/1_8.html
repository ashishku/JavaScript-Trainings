Expressions

  • Primary Expressions 
      // Litrals
  1.23            // A number literal
  "hello"         // A string literal
  /pattern/       // A regular expression literal
  // Keywords
  false           // Evaluates to the boolean false value
  null            // Evaluates to the null value
  this            // Evaluates to the "current" object
  // variables
  i               // Evaluates to the value of the variable i.
  sum             // Evaluates to the value of the variable sum.
  undefined       // undefined is a global variable, not a keyword like null.
  • Object and Array Initializers 
      []                                  // An empty array: no expressions inside brackets means no elements
  [1+2,3+4]                           // A 2-element array. First element is 3, second is 7
  var matrix = [[1,2], [3,4], [5,6]]; // 2 dimension array
  var sparseArray = [1,,,,5];         // spare array with undefined values
  
  var p = { x:2.3, y:-1.2 };          // An object with 2 properties
  var q = {};                         // An empty object with no properties
  q.x = 2.3; q.y = -1.2;              // Now q has the same properties as p
  
  // Nested Objects and nested expresion 
  var side = 1;
  var square = {
    "upperLeft" : { x: p.x, y: p.y },
    "lowerRight": { x: p.x + side, y: p.y + side}
  };
  • Function Definition Expressions 
      // This function returns the square of the value passed to it.
  var square = function(x) { return x * x; }
  • Property Access Expressions 
      expression . identifier
  expression [ expression ]

  var o = {x:1,y:{z:3}};      // example object
  o.x                         // 1: property x of expression o
  o.y.z                       // 3: property z of expression o.y
  o["x"]                      // 1: property x of object o

  var a = [o,4,[5,6]];        // example array that contains the object
  a[1]                        // 4: element at index 1 of expression a
  a[2]["1"]                   // 6: element at index 1 of expression a[2]
  a[0].x                      // 1: property x of expression a[0]
  • Invocation Expressions 
      f(0)                // f is the function expression; 0 is the argument expression.
  Math.max(x,y,z)     // Math.max is the function; x, y and z are the arguments.
  a.sort()            // a.sort is the function; there are no arguments.
